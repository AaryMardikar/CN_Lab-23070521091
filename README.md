# Computer Networks Lab - Semester 5

This repository contains all the lab assignments, programs, and study files for the 5th Semester Computer Networks (CN) course.

---

## Lab Experiments

This table outlines all the practicals and their corresponding experiments as per the curriculum.

| Practical No. | Experiments & Concepts Covered |
| :--- | :--- |
| **1** | Study of Basic elements of computer networking with details of networking devices. |
| **2** | Packet Analysis using a packet tracer tool with detailed analysis of Application Layer packets. |
| **3** | Socket Programming for Transport Layer packets (TCP and UDP). |
| **4** | Analysis of Application layer packet (DNS) using DNS commands & Analysis using a packet tracer tool. |
| **5** | Implement Sliding Window Protocols. |
| **6** | Implement Traffic shaping Algorithms. |
| **7** | Implement Distance Vector Algorithm. |


---

##  How to Use

This section provides general instructions on how to compile and run the programs in this repository.

### Prerequisites

Make sure you have a C / C++ / Java / Python compiler or interpreter installed, as well as any necessary packet tracer tools (like Wireshark).

### Compilation & Execution

**For C/C++ Programs:**
```bash
# Compile the program
gcc program_name.c -o program -lm

# Run the program
./program
```
